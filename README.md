## What I Build

- Agent systems that remember context, progress, decisions, and reusable experience
- Browser automation workflows that are persistent, inspectable, and safe to operate
- Educational AI tools that turn knowledge into visual, interactive, and memorable stories

## Selected Work

### [Session-engram](https://github.com/ZyanWan/Session-engram)

Cross-session memory for AI coding assistants. It preserves context, task progress, reusable experience, timelines, and knowledge graphs across conversations.

`Python` `Markdown` `Hooks` `Knowledge Graph` `AI Memory`

### [Patchright-Agent-Skills](https://github.com/ZyanWan/Patchright-Agent-Skills)

Reusable real-browser automation skills for AI Agents. It packages persistent sessions, page diagnostics, controlled actions, evidence capture, and workflow references into reusable skill modules.

`Python` `Patchright` `Browser Automation` `Agent Skills` `Workflow`

### [Zyan](https://github.com/ZyanWan/Zyan)

Full-stack AI Agent platform prototype with conversational AI, MCP protocol integration, knowledge base, RAG vector search, multi-model switching, and visual workflow orchestration.

`React` `TypeScript` `Fastify` `PostgreSQL` `RAG` `MCP`

### [Hanzi-Etymology-Animation](https://github.com/ZyanWan/Hanzi-Etymology-Animation)

AI-powered Chinese character evolution video generator for language learning. It combines etymology data, glyph processing, TTS narration, and AI video generation for visual storytelling.

`Python` `SVG` `TTS` `AI Video` `EdTech`
